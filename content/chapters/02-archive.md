# L'archive basse

Sous la bibliotheque, un second lieu dormait. Les marches descendaient trop longtemps. A chaque palier, Galgorat trouvait la meme marque gravee dans la pierre: un cercle ouvert, une clef sans dents.

Il compta quarante-sept portes avant de comprendre. L'archive ne gardait pas les livres termines. Elle gardait les versions abandonnees, les chapitres refuses, les fins que personne n'avait ose prononcer.

```txt
registre: basalte
entree: non classee
risque: lecture recursive
ordre: ne pas corriger les marges rouges
```

La derniere porte cedait deja.
